This project is creating a Library database and allowing users to edit that database or to view the information regarding that database. The users are given a set of options of either adding a book to the database whether it is a audio book or a printed book along with the given information of the book (title, author, genre, pages/duration(minutes)). They can sort the list of books that they either create or are given alphebatically by title, they can save or load the file of all the books. They can also view the statistics of the books, or they can end the program. 
The printed book and audio book have the difference of either amount of pages or their duration in minutes, which differenciate them through sorting, as well as through cost, as printed books cost more than audio books.
